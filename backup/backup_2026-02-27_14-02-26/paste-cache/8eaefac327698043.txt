  📌 详细介绍

  1. BBDown ⭐ 命令行首选
  - 技术：.NET Tool 形式发布
  - 特点：智能接口切换、高清画质、批量下载
  - 需要：ffmpeg/mp4box 混流支持
  - 适用：技术用户、自动化脚本

  2. DownKyiCore（哔哩下载姬跨平台版）
  - 技术：AvaloniaUI + Aria2 + FFmpeg
  - 特点：8K/HDR/杜比视界、GUI界面、扫码登录
  - 支持：Windows/macOS/Linux 全平台
  - 适用：普通用户、桌面应用

  3. BilibiliDown
  - 技术：Java 跨平台
  - 特点：稍后再看、收藏夹、UP主批量下载
  - 适用：批量下载需求

  ---
  🔧 核心技术原理

  ┌──────────┬──────────────────────────────────────┐
  │  技术点  │                 说明                 │
  ├──────────┼──────────────────────────────────────┤
  │ DASH协议 │ 音画分离存储，视频流和音频流分别请求 │
  ├──────────┼──────────────────────────────────────┤
  │ Wbi签名  │ URL参数加密，需要理解加密原理        │
  ├──────────┼──────────────────────────────────────┤
  │ SESSDATA │ 用户认证信息，获取会员权限           │
  ├──────────┼──────────────────────────────────────┤
  │ .m4s分片 │ DASH流媒体加密分片，需解密合并       │
  ├──────────┼──────────────────────────────────────┤
  │ HEVC编码 │ 比AVC高效，同等画质减少50%体积       │
  └──────────┴──────────────────────────────────────┘
